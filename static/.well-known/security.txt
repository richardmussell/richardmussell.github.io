# Reporting Process:
# 1. Contact via email or contact form
# 2. Provide detailed description of the vulnerability
# 3. Allow reasonable time for remediation before public disclosure
#
# Scope:
# - This website (richardmussell.github.io)

# - Third-party services (report to respective providers)

# - Issues requiring physical access
