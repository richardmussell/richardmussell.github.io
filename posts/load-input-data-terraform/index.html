How to Load Input Data from a File in Terraform?

Jul 1, 2025 1 min read

Learn different methods to load external data into Terraform configurations. Explore file(), fileexists(), and data sources. Understand when to use each approach and how to structure your data files for maximum flexibility.

Loading Data from Files

Terraform provides several functions for reading file contents:

file() function

locals {
  config = file("${path.module}/config.json")
}

fileexists() function

locals {
  has_config = fileexists("${path.module}/config.json")
}

External Data Source

For more complex scenarios, use the external data source to execute scripts and read their output.
